# Backend Documentation

This folder documents how to run, test, maintain, inspect, and extend the
Invora backend.

- `project-runner.md`: step-by-step local setup and run flow
- `commands.md`: daily development commands
- `testing.md`: test strategy and test commands
- `architecture.md`: modular monolith structure and layer rules
- `swagger.md`: Swagger UI, ReDoc, OpenAPI JSON, and auth verification notes
- `progress.md`: current implementation status and pending modules
