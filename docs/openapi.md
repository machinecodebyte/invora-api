# OpenAPI

OpenAPI JSON is generated by the running FastAPI application:

```text
http://127.0.0.1:<API_PORT>/openapi.json
```

Swagger UI and ReDoc use the same configured port:

```text
http://127.0.0.1:<API_PORT>/docs
http://127.0.0.1:<API_PORT>/redoc
```

See [Swagger and OpenAPI](swagger.md) for token authorization and API-group
verification.
